# self-hosted-ai-development-environment

This is a system I have created, implemented, and use to aid in development, free of charge and securely. 

This system is built using the following softwares:
- Debian 12 Linux
- Docker
- CasaOS
- Ollama (various models detailed in workflows.md)
- Open WebUI
- Roo Code
- Tailscale

## Objectives

- Eliminate API token costs
- Have complete customisation control
- Support programming projects
- Create project specific AI workflows
- Experiment with optimising workflows useing AI
