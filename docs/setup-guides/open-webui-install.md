# Open WebUI Installation

1. Open CasaOS
2. Go to App Store
3. Search: **Open WebUI**
4. Install container
5. Open via CasaOS
6. (optional) set port to:
```bash
# direct access
http://<server-ip>:3000

# if using tailscale
http://100.x.x.x:3000
```
7. Set up Open WebUI local account
8. Go to settings
9. Set backend URL to:
```bash
# direct access
http://<server-ip>:11434

# if useing tailscale
http://100.x.x.x:11434
```
10. Verify models are available to use
